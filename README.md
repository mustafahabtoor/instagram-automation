# Instagram Comment-to-DM Automation Tool

A full-stack web application that automates Instagram comment replies and direct messages based on keywords.

## One-Click Deploy
Deploy your own instance to Railway instantly:

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/new/template?template=https://github.com/mustafahabtoor/instagram-automation)

## Features:
- Official Instagram Graph API integration.
- Clean and simple web dashboard for settings and campaigns.
- Automatic comment deduplication to prevent double-firing.
- Webhook security validation.
